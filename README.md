#### Локальная раработка

1. Клонируем репозиторий
2. Устанавливаем зависимости: `yarn`
3. Скопировать корневой файл `env-example` в созданный `.env`
4. В корневом файле `.env` сгенерировать JWT
3. Запустить development: `yarn start`
